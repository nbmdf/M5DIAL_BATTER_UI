# M5Dial Energy Monitoring System  
Battery / PV / GenSet / Load Modules (ESP32-S3, MQTT, JSON)

This repository contains **four IoT firmware modules** designed for the M5Dial (ESP32-S3) device.  
Each module communicates with the cloud via **MQTT + JSON**, forming a complete renewable energy monitoring system.

Modules Included:
-  **Battery Module**
-  **PV (Solar) Module**
-  **GenSet (Generator Set) Module**
-  **Load Module**
